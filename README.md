# 오늘 알림장

학교에서 바로 열어 사용할 수 있는 단일 HTML 기반 알림장 프로그램입니다.

## 구성 파일

- `index.html` : 메인 실행 파일
- `README.md` : 사용 및 GitHub 업로드 안내
- `.gitignore` : 불필요 파일 제외용

## 실행 방법

1. `index.html` 파일을 더블클릭합니다.
2. 브라우저에서 바로 실행됩니다.
3. 별도 설치가 필요 없습니다.

## 주요 기능

- 날짜 자동 생성
  - 주말과 등록된 법정공휴일 자동 제외
  - 날짜는 입력 화면에 보이지 않고, 완료 확인 및 공유 문구에만 반영
- 오늘 공부한 내용
  - 기본 5교시
  - 교시 추가/삭제 가능
  - 각 항목 글자색 선택 가능
- 과제 / 안내
  - 소항목 추가/삭제 가능
  - 각 항목 글자색 선택 가능
- 가치 사전
  - 가치 낱말 선택 시 뜻과 실천 방법 자동 표시
- 기록자 관리
  - 기록자 목록 추가/선택/삭제 가능
  - 내부 저장용으로 사용
- 자동 저장
  - 새로고침 후에도 입력 내용 유지
- 완료 전 확인 모달
  - 최종 화면 확인 후 저장 및 공유 가능
- 카카오톡 공유
  - 모바일 공유창 또는 문구 복사 방식 지원

## GitHub 업로드 방법

### 방법 1: 웹에서 바로 업로드

1. GitHub에서 새 저장소를 생성합니다.
2. `Add file` → `Upload files`를 누릅니다.
3. 이 폴더 안의 파일들을 모두 올립니다.
4. `Commit changes`를 누릅니다.

### 방법 2: GitHub Pages로 배포

1. 저장소에 이 파일들을 업로드합니다.
2. 저장소의 `Settings` → `Pages`로 이동합니다.
3. `Build and deployment`에서 Source를 `Deploy from a branch`로 설정합니다.
4. Branch를 `main` / `/root`로 선택하고 저장합니다.
5. 잠시 후 배포 주소가 생성됩니다.
6. 배포 주소에서 `index.html`이 바로 실행됩니다.

## 사용자 수정 포인트

### 1. 공휴일 수정
`index.html` 내부의 아래 상수를 수정하면 됩니다.

```js
const HOLIDAYS_BY_YEAR = {
  "2026": [
    "2026-01-01"
  ]
};
```

### 2. 가치 사전 수정
`index.html` 내부의 `VALUE_DICTIONARY` 배열을 수정하면 됩니다.

### 3. 기본 글자색 수정
`index.html` 내부의 `DEFAULT_TEXT_COLOR` 값을 바꾸면 됩니다.

## 주의 사항

- 브라우저 보안상 특정 카카오톡 대화방을 자동으로 직접 지정해 보내는 기능은 지원되지 않습니다.
- 대신 공유창을 열거나 문구를 복사해 붙여넣는 방식으로 사용합니다.
- 저장 데이터는 사용 중인 브라우저의 로컬 저장소(localStorage)에 저장됩니다.
